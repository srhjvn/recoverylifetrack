## Purpose
Create a structured path to:
1. obtain a formal mental health diagnosis and supporting documentation,
2. apply for SSI and/or SSDI through the Social Security Administration,
3. contact the lender to discuss hardship or payment options,
4. seek legal or nonprofit support if representation or debt-management help is needed.

---

## Project Objective
Build a complete and organized support file that improves readiness for disability benefits review, reduces financial risk, and creates a documented path for next-step support.

---

## Definition of Done
- formal mental health evaluation is completed
- relevant medical records are requested and organized
- symptom/function impact summary is drafted
- SSI and/or SSDI application is submitted
- copies of SSA forms and notices are saved
- lender is contacted and payment-relief options are documented
- legal aid and/or nonprofit support options are identified
- all key contacts, deadlines, and documents are stored in one place

## Key Workstreams

### 1. Medical Diagnosis and Documentation
**Goal:** Obtain a formal diagnosis and collect records supporting functional limitations.

**Core Tasks**
- identify provider for formal mental health evaluation
- schedule appointment
- request copies of diagnosis, treatment notes, medication history, and related records
- create provider contact list
- draft symptom/function timeline
- summarize how symptoms affect daily living and ability to work

**Deliverables**
- formal diagnosis
- medical records packet
- provider list
- symptom/function summary

**Dependencies**
- provider availability
- existing treatment history
- ability to request and receive records

---

### 2. SSA Disability Application
**Goal:** Prepare and submit a complete SSI and/or SSDI application.

**Core Tasks**
- determine likely application path: SSI, SSDI, or both
- gather personal identification and work history
- gather provider, treatment, and medication information
- complete pre-application checklist
- complete SSA application
- complete Adult Disability Report or equivalent supporting forms
- save submission confirmations and copies

**Deliverables**
- submitted application
- claim file with copies of forms
- application tracker
- deadline/notice log

**Dependencies**
- medical documentation
- work history details
- SSA account access or appointment availability

---

### 3. Lender Hardship Communication
**Goal:** Contact lender early and document available hardship options.

**Core Tasks**
- identify lender contact channel
- prepare brief hardship summary
- call lender and request hardship/payment-relief discussion
- ask about extension, deferment, modified payment, due-date change, or temporary hardship options
- request written confirmation of any offer
- document call details and next steps

**Deliverables**
- lender call log
- summary of options offered
- written hardship terms if available
- follow-up deadline list

**Dependencies**
- account information
- lender response timeline
- current delinquency status

---

### 4. Legal or Nonprofit Support
**Goal:** Identify outside support for representation, appeal help, or debt management.

**Core Tasks**
- identify local legal aid and disability advocacy resources
- identify nonprofit credit counseling / debt management options
- review whether representation is needed now or later
- contact at least one support organization
- document intake requirements and next steps

**Deliverables**
- support resource list
- consultation/contact log
- representation decision notes
- debt-management referral notes if applicable

**Dependencies**
- local availability
- income/resource eligibility
- claim complexity or lender pressure

## Phase Structure

### Phase 1: Immediate Stabilization
**Focus:** Start the highest-risk and longest-lead items.
- schedule mental health evaluation
- begin requesting records
- contact lender
- create project folder and tracker

### Phase 2: Documentation Build
**Focus:** Gather and organize evidence.
- complete provider list
- gather records received
- write symptom/function summary
- compile work history and treatment history

### Phase 3: Application Submission
**Focus:** File benefits claim.
- complete SSA forms
- submit SSI/SSDI application
- save all confirmations
- log pending follow-ups

### Phase 4: Escalation and Support
**Focus:** Add support if needed.
- contact legal aid or advocate
- contact nonprofit counseling if debt pressure rises
- prepare for follow-up requests, reconsideration, or appeal if necessary
