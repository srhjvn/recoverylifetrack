# Disability Benefits and Financial Stabilization

This folder contains the minimal supporting markdown files for the project.

## Files
- `medical-documentation-checklist.md` — organizes medical records, provider info, and symptom/function evidence
- `ssa-application-workflow.md` — step-by-step application workflow and case-tracking guide
- `lender-hardship-log.md` — call log and option-comparison file for lender hardship communication
- `support-resources.md` — resource directory and outreach log for legal, advocacy, and nonprofit support
- `master-deadline-and-contact-tracker.md` — top-level tracker for contacts, deadlines, follow-ups, and risks



## Companion Files
- `medication-list.md` — current and past medication tracker for appointments, forms, and treatment history
- `application-notes.md` — staging area for claim notes, draft answers, and submission follow-up
- `legal-aid-contact-log.md` — outreach log for legal aid, advocacy, and debt-support contacts

## Cross-File Support Pieces
- `document-index.md` — master map of what documents exist, where they are stored, and what is still missing
- `appointment-log.md` — tracker for appointments, evaluations, calls, and follow-up actions
- `financial-hardship-summary.md` — reusable hardship summary for lenders, legal aid, and support intake


## Final Execution Support Files
- `ssa-notice-log.md` — source-of-truth log for SSA mail, deadlines, and required responses
- `task-board.md` — simple action board for now / next / waiting / blocked / done
- `phone-call-script-notes.md` — reusable scripts and note space for SSA, lenders, providers, and support calls

## Suggested Use Order
1. update `master-deadline-and-contact-tracker.md`
2. list what exists in `document-index.md`
3. work through `medical-documentation-checklist.md`
4. fill out `provider-list.md`, `records-request-log.md`, and `appointment-log.md`
5. draft `symptom-function-summary.md`, `medication-list.md`, and `work-history.md`
6. summarize current pressure in `financial-hardship-summary.md`
7. prepare from `application-notes.md` and `ssa-application-workflow.md`
8. document lender communication in `lender-hardship-log.md`
9. track outside support in `support-resources.md` and `legal-aid-contact-log.md`
10. log SSA mail in `ssa-notice-log.md`
11. manage next actions in `task-board.md`
12. prepare important calls in `phone-call-script-notes.md`
