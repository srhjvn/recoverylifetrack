## Purpose
Create a clean work-history source file that can be reused for SSA forms, appointments, benefit discussions, and personal reference.

---

## Definition of Done
- relevant work history is listed in order
- job titles, employers, and dates are captured as best known
- job duties are summarized in plain language
- work demands and barriers are visible
- reasons work became unsustainable are documented clearly enough to support form completion

---

## How to Use This File
- start with the most recent work and move backward
- use approximate dates if exact dates are not yet available
- focus on what the work actually required, not just job titles
- keep notes factual and plain-language
- use this file to reduce inconsistencies across SSA forms and conversations
