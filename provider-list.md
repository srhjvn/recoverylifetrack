## Purpose
Create a single source of truth for every medical, mental health, hospital, therapy, and support provider connected to the claim or documentation process.

---

## Definition of Done
- all known providers are listed
- missing provider details are flagged for follow-up
- treatment date ranges are captured as best known
- records-request status is visible for each provider
- the file can be used as a source document for SSA forms, phone calls, and follow-up requests

---

## How to Use This File
- add every provider you can remember, even if some details are incomplete
- use one row per provider or facility
- update this file before filling out forms so dates and names stay consistent
- mark unknown details instead of leaving them invisible
- pair this file with `records-request-log.md` when asking for records

---

## Status Key
- **Not Started** = provider identified but no outreach yet
- **Info Partial** = some details still missing
- **Requested** = records request sent
- **Received** = records received and stored
- **Follow-Up Needed** = request sent but still incomplete
- **Closed** = no further action currently needed
